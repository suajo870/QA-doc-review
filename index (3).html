GitHub Pages 배포 가이드

팀원 모두가 같은 URL로 접속할 수 있게 배포하는 방법

🚀 배포 순서 (약 10분)
1

GitHub 계정 만들기

github.com/signup에서 무료 계정을 생성하세요. 팀 공용 계정을 권장해요.

2

새 Repository 만들기

로그인 후 + → New repository 클릭
이름: qa-doc-review / Visibility: Public → Create repository

3

index.html 파일 업로드

다운로드한 index.html을 Repository 페이지로 드래그하거나 uploading an existing file을 클릭해 업로드하세요.

4

GitHub Pages 활성화

Repository → Settings → Pages
Source: Deploy from a branch / Branch: main / (root) → Save

5

팀원에게 URL 공유

1~2분 후 아래 주소로 접속 가능해요.

https://[GitHub계정명].github.io/qa-doc-review
⚠️ 데이터 공유 주의: 현재는 각 팀원 브라우저에 개별 저장됩니다. 팀원 간 실시간 공유가 필요하면 Google Sheets 연동을 추가할 수 있어요.
